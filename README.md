# 📊 Customer Churn Analysis Project

## 📌 Overview

This is an end-to-end data science project focused on analyzing customer churn in a telecom company.
The goal is to identify key factors affecting customer churn and provide actionable insights.

---

## 🛠 Tools & Technologies

* Python (Pandas, Matplotlib, Seaborn)
* Power BI
* Excel (Data Cleaning & Preprocessing)
* Machine Learning (Classification Model)

---

## 📂 Project Components

### 📊 Dashboard

* Interactive Power BI dashboard
* Visualizes churn trends across contract types, demographics, and billing

### 📈 EDA (Exploratory Data Analysis)

* Data cleaning performed using Excel
* Handled missing values, duplicates, and inconsistencies
* Data analysis using Python

### 🤖 Machine Learning

* Built a churn prediction model
* Identified high-risk customers

### 📄 Product Manager Insights

* Provided business recommendations to reduce churn
* Suggested strategies for customer retention

---

## 📊 Key Insights

* Month-to-month contracts have the highest churn
* Customers with high monthly charges are more likely to churn
* Customers with low tenure are at higher risk
* Payment method influences churn behavior
